What is BonoBooks?

BonoBooks is a full-stack application for peer-to-peer textbooks rentals. Catered towards college students who want to earn some extra income on the side by renting out their old course textbooks, or those who want to save some money on cheaper priced textbook rentals from another student. With the support of marketing, booths, and other advertising efforts, we attracted over 190 users and over 50 rental exchanges through the app.


The application is built using Flutter/Dart, an open-source framework by Google, to allow multi-platform support. In this case, we utilized Flutter to easily deploy to both iOS and Android. For the database and storage, Firebase was implemented to allow user authentication, data storage and retrieval, and analytics. Stripe for rental payments and Algolia to allow full-text search for rentals available within our app's marketplace. Finally, Javascript and Firebase Cloud Functions was used for any scheduled tasks or tasks that required external resources.


As a team of 4, the entire software development life cycle was planned and executed all from the design and ideation of the application to the implementation and refinement of the product.
